# **App Name**: NeetFlow

## Core Features:

- Immersive Subject Gateway: Interactive landing page featuring floating 3D-styled cards for Biology, Physics, and Chemistry that animate upon entry.
- Personalized Onboarding Flow: Clean, focused name entry screen with glowing input states and validation-driven button activation.
- Intelligent Persistence: State management system using local storage to recognize returning users and automatically navigate to their personalized welcome dashboard.
- Dynamic Champion Dashboard: A customized welcome screen for the 'NEET CHAMP' showing availability updates with smooth entrance transitions.
- AI Study Spark Tool: An AI tool that generates personalized motivational prompts and daily scientific 'pro-tips' to keep students focused.

## Style Guidelines:

- A sophisticated dark theme inspired by nocturnal deep-focus sessions. Primary: Electric Violet (#7C5CFF) for vibrancy; Background: Midnight Abyss (#121018), a deep desaturated purple; Accent: Tech Blue (#7393FF) for interactive highlights.
- A modern font pairing using 'Space Grotesk' for high-impact headlines and the bold 'NEET' branding, combined with 'Inter' for UI labels and readable body text to ensure a tech-forward, objective feel.
- Minimalist centered layouts with glassmorphic elements and high-contrast subject cards to maintain extreme focus and a premium educational experience.
- Micro-interactions focused on depth: floating floating-book movements on the X/Y axes, smooth SVG morphs for transitions, and a slight Z-axis zoom effect on entry button clicks.
- Fine-lined, monochromatic iconography for Biology, Physics, and Chemistry, emphasizing a professional and academic aesthetic.
